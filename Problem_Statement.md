# Day 8 — Bayesian Inference Engine

## Problem Statement
Build a Bayesian inference engine from scratch that updates beliefs given observed evidence, without using any statistics libraries.

## Core Questions
- How do we update beliefs when new evidence arrives?
- What is the difference between discrete and continuous inference?
- How does a conjugate prior eliminate numerical integration?

## Requirements
- Discrete Bayesian inference over explicit hypothesis buckets
- Beta-Binomial conjugate model for continuous bias estimation
- Visualizations: prior/likelihood/posterior, sequential updating, Beta PDF evolution, credible intervals
- All math from scratch — no scipy, no stats libraries
